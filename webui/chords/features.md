# Features 1.0

KEY - E opens edit of chords
KEY - S picks a song (known or simple jams)
KEY - C toggles chords
KEY - D toggles drums
KEY - R random progression (better random generator)
KEY - space toggles play / stop
KEY - dedicated tap tempo button, such as T
KEY - swing support: down sets straight feel, left swings towards beat one way, right the other way
  - 7 settings: straight, plus three on each side of the beat
- eighth note option for metronome / drums
- Quick changes / submeasures for a chord
- Immediate, quick, or gradual changes between pieces
- sections
- Strong visual indicator on the downbeat of a chord change
- heavier kick drum on the 1 & 3
- transpose mechanism (e.g. up / down)
- Visualizations
  - different complexity levels
    - terminal: C___|D___|.___|G_D_C
    - simplified
    - animated metronome / drum stick
    - animated strumming hand
  - different device classes
    - phone: touch screen, small, simplified
    - tablet: touch screen, larger
    - laptop
    - desktop
- Library:
  - look up thousands of songs (youtube, track library, internet)
  - edit / author and save song
  - Know the chord structure, tempo, and composition of all music
  - import the chords and drum pattern for any pop song
  - melody lines
- Rhythm:
  - more pre-installed patterns
  - longer patterns => multiple measures
  - attack / decay
- Chords:
  - minor and 7th chords
  - arpeggios
  - chord diagrams
  - change voicings of chords
  - sample each of the chords from a electronic piano over midi
  - sample library from guitars
  - voice as strings (violins - dynamics)
- Drums:
  - integration with drum machine / samples
  - editable drum patterns
  - drum patterns
  - tap tempo
